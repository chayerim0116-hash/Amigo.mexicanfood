5-2. Hero Banner 
Hero 배너는 메인 페이지의 헤더 아래에 위치한다. 
메인 페이지의 헤더 바로 아래에 가로로 긴 Hero 배너 영역을 만들어줘. 
이 배너는 화면 전체 높이를 꽉 채우는 풀스크린 Hero가 아니라 쇼핑몰 메인 페이지 상단에 들
어가는 와이드한 메인 비주얼 배너야. 전체 분위기는 깔끔하고 차분한 여성 패션 쇼핑몰 느낌
으로 만들어줘. 
hHero 배너는 총 3장의 이미지로 구성해줘. 
사용 이미지: 
./img/banner/banner1.png 
./img/banner/banner2.png 
./img/banner/banner3.png 
각 이미지는 하나의 슬라이드로 배치하고 슬라이드 전체를 감싸는 track 요소를 가로 방향으
로 움직여서 다음 배너로 전환해줘. 
h구조는 아래처럼 만들어줘 
main-visual: 전체 Hero 배너 영역 
visual-track: 3개의 슬라이드를 가로로 나열하는 영역 
visual-slide: 각각의 배너 이미지와 텍스트가 들어가는 개별 슬라이드 
visual-indicator: 배너 아래에 들어가는 진행 바와 현재 순서 표시 영역 
hHero 배너 전체 영역은 이렇게 잡아줘. 
main-visual은 position: relative로 두고 높이는 40.3vw로 설정해줘. 최소 높이는 360px로 
유지하고 넘치는 영역은 보이지 않게 overﬂow: hidden 처리해줘. 배경색은 #f4f1ed로 해줘.
슬라이드 이미지는 배너 영역을 꽉 채우게 보여줘. 현재 결과와 동일하게 이미지가 배너 프레
임 안을 가득 채우는 방식이어야 하므로 object-ﬁt은 cover를 사용해줘. 이미지 위치는 중앙
